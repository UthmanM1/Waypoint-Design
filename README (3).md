## What is included

- Cinematic scroll sequence: airplane window, chaos scene, Cappadocia sunrise with balloons
- 10-step onboarding flow with emotional intent selection and style tags
- Pricing page with monthly/annual toggle
- AI planner chat with hotel suggestion cards, typing indicator, and quick-reply chips

All screens are fully responsive and built in the exact aesthetic described in the brief: dark backgrounds, editorial typography, gold accent system.
